How to Use This Page

This page helps you test your Google OAuth configuration:

  1. Click the "Initiate Google Login" button to start the OAuth flow
  2. You'll be redirected to Google's login page
  3. After successful authentication, you'll be redirected back to your application
  4. Check the URL parameters and console for any errors

Troubleshooting

If you see a "redirect_uri_mismatch" error:

  1. Go to the Google Cloud Console
  2. Find your OAuth 2.0 Client ID
  3. Add the redirect URI shown above to the list of authorized redirect URIs
  4. Save your changes and try again
